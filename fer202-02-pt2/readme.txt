Personal Budget Management Application
Subject code: FER202 - FALL 2025

Installed Packages:
- react
- react-dom
- react-router-dom
- @reduxjs/toolkit
- react-redux
- axios
- bootstrap
- react-bootstrap
- json-server

How to run:
1. Open a terminal and start the JSON Server:
   npx json-server --watch db.json --port 9999

2. Open another terminal and start the React application:
   npm start

Features implemented:
- Authentication (Login/Logout) with validation.
- Monthly Dashboard with real-time updates.
- Total Expenses calculation and formatting (VND).
- Expense Management (CRUD: Create, Read, Update, Delete).
- Filter by Category.
- Responsive design with React Bootstrap.
- State management using Redux Toolkit.
